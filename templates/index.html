Important: Enable Message Content Intent

For commands to work, you must enable Message Content Intent in your bot's settings at the Discord Developer Portal. Go to Bot ‚Üí Privileged Gateway Intents ‚Üí Enable "Message Content Intent".

How to Use
  1. Create a bot application in the Discord Developer Portal
  2. Enable Message Content Intent: In the Bot section, scroll down to "Privileged Gateway Intents" and enable "Message Content Intent"
  3. Copy your bot token from the Bot section
  4. Paste the token in the input field above
  5. Click "Start Bot" to begin running your Discord bot
  6. Invite your bot to a server and try the available commands
